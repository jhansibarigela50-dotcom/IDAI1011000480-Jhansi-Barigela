Unit 1: AI-Based Smart Solutions: Designing Chatbots for Real-World Challenges

This NutriWell chatbot aids people in building up their wellness through fitness guidance and tracking. The senario this follows is that of an employee in a health and wellness company that aims to build an intelligent chatbot using their internal AI development team. Through the use of intents and entities the chatbot is able to create a customized dietary plan based on food restrictions, reccomend an approporiate meal plan, and give fitness and hydration reminders. The intents help the conversation flow while the entities allow inputs of unique responses. It begins with the welcome intent where the user is greeted and asked for their name to start the conversation. After the name is provided, the name intent will come into play. This intent has an "any" entity which allows different names to be provided by the user.
